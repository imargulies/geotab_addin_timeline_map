# Geotab Minute-by-Minute GPS Tracker Add-In

## Installation Instructions

1. **Login to MyGeotab**
   - Go to: Administration → System → System Settings

2. **Navigate to Add-Ins**
   - Click "Add-Ins" in the left menu

3. **Create New Add-In**
   - Click the "+" button in the top right corner
   - Paste the contents of `configuration.json` into the dialog
   - Click "OK" to create the add-in

4. **Upload Files**
   - Click on your newly created add-in in the list
   - Click the "Files" tab
   - Upload these files (click "+" for each):
     - timeline.html
     - timeline.css
     - timeline.js

5. **Save**
   - Click "OK" to save the add-in

6. **Launch**
   - Go to: Activity → Minute by Minute GPS
   - The add-in will launch in the main window

## Features

- ✅ Select any vehicle from your fleet
- ✅ Choose date/time range
- ✅ Minute-by-minute location tracking
- ✅ Real addresses from Geotab geocoding
- ✅ Click any minute to see location on map
- ✅ View trail showing ±5 minutes around selected point
- ✅ Speed data for each minute
- ✅ Automatic map fitting to route

## Usage

1. Select a vehicle from the dropdown
2. Choose start and end date/time
3. Click "Load Timeline"
4. Click any minute in the left panel to view on map
5. Map shows the exact location with a trail of nearby points

## Files Included

- configuration.json - Geotab add-in configuration
- timeline.html - Main HTML structure
- timeline.css - Styling
- timeline.js - JavaScript logic and Geotab API integration
